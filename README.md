# SwarmShieldTech Cloud – Corporate Technical Website

This is the official technical website for **Swarm Shield Tech**, deployed at:

https://swarmshieldtech.cloud

Tech stack:

- React + Vite
- TailwindCSS
- Cloudflare Pages
- Lucide Icons
- Fully responsive, dark enterprise theme

Deploy with:

1. Push repo to GitHub  
2. Cloudflare Pages → Create project  
3. Build command: `npm run build`  
4. Output directory: `dist`
